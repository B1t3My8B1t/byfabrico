## Overview

Static marketing website for **Fabrico Design & Build** — a high-end construction and interior design company operating across London and Berkshire. The site showcases project portfolios, services, and handles client enquiries via an integrated contact form.

## Sections

The site is a single `index.html` with anchor-linked sections:

- `#top` — Hero (Chelsea SW3 hero image, headline, CTAs, stats)
- `#about` — Company story, credentials (5-star / Insured / CHAS)
- `#services` — 6-service icon grid + service selector pills
- `#enquire` — Service quick-select (feeds into form)
- `#projects` — Portfolio groups: Chelsea · Mayfair · SW5 · Cookham · Renovations · New Builds · Structural · Before & After
- `#process` — 5-step process strip
- `#contact` — Contact details + Web3Forms enquiry form
- Footer — Nav, social, SEO copy
